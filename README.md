# Sombre Foundry Compendium Module

Module pour **Foundry VTT** permettant la création et le peuplement automatique de compendiums pour le jeu de rôle Sombre, la peur comme au cinéma de **Johan Scipion**.

Ce module génère automatiquement des compendiums contenant :
- des **traits**
- des **personnalités**

Ces éléments deviennent ensuite disponibles directement dans les fiches de personnages du système Sombre.

## Installation

### 1. Télécharger le module

Téléchargez l’archive `.zip` du module.

---

### 2. Configurer les données

Avant l’installation, modifiez les fichiers suivants :

- `traits.json`
- `personnalities.json`

Vous pouvez y ajouter les traits et personnalités de votre choix.

⚠️ **Important :**
- Ne modifiez **pas** le type des champs
- Ne modifiez **pas** les intitulés des champs
- Ajoutez uniquement vos propres entrées en respectant la structure existante

---

### 3. Installer le module sur votre serveur Foundry

Installez ensuite le module sur votre serveur Foundry VTT.

La méthode dépend de votre type d’hébergement :
- hébergement local
- serveur dédié
- hébergement mutualisé
- services d’hébergement Foundry

---

### 4. Activer le module

Dans Foundry :
1. Ouvrez un monde utilisant le système **Sombre**
2. Activez le module depuis le menu des modules

---

## Fonctionnement

Une fois activé, le module crée automatiquement deux compendiums :

- un compendium pour les **personnalités**
- un compendium pour les **traits**

Les items créés seront ensuite accessibles directement dans les fiches de personnages.
